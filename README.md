# FEND Capstone-Travel App

## Overview
This project a travel application.The project will include a simple form where you enter the location you are traveling to and the date you are leaving.

## API's used
 -Geonames
 -Pixabay
 -weatherbit

## How to start
 - npm install
 - npm run build-prod
 - npm run start
 - npm run build-dev
 for testing:
 - npm run test
